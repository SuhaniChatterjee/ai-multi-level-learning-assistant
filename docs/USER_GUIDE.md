# Cognitive Assistant - User Operations Guide

Welcome to the AI Learning Assistant!

## Operating the App

1. **Deploy:** Start the service by running `./run.sh` from the root directory.
2. **Access UI:** Open `http://localhost:8501`.
3. **Upload Material:** Upload your academic PDF into the `Document Ingestion` field.
4. **Choose Depth:** Select from `ELI5`, `College`, or `Interview` complexity.
5. **Prompt:** Enter your question into the Query Engine and generate your custom UI response!

## Requirements
You must possess a free Hugging Face API key, and set it via the `HUGGINGFACEHUB_API_TOKEN` environment variable.
